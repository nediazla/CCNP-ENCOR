# CCNP ENCOR 350-401 — Apuntes Personales y en español 

Bienvenido a mi repositorio de apuntes personales para la certificación **Cisco CCNP Enterprise Core (ENCOR 350-401)**.  
Este repositorio forma parte de mi preparación profesional en redes empresariales, seguridad, wireless y automatización.

Los apuntes están organizados por capítulos, siguiendo la estructura oficial de estudio, e incluyen resúmenes técnicos, conceptos clave, comandos Cisco IOS/XE, fórmulas, diagramas y notas prácticas para laboratorio.

---

## Objetivo del repositorio

Este proyecto recopila de forma ordenada mis **notas de estudio**, resúmenes y explicaciones claras sobre los temas más importantes del examen ENCOR:

- Routing avanzado (OSPF, EIGRP, BGP)  
- Switching Enterprise (STP, RSTP, MST, EtherChannel, VLANs)  
- Wireless (RF, modulación, roaming, arquitectura con WLC, RRM)  
- Seguridad (NAC, AAA, control de acceso)  
- Network Assurance  
- QoS  
- Virtualización y overlays (GRE, IPsec, DMVPN, VXLAN, LISP)  
- Automatización y model-driven telemetry  
- Servicios de infraestructura (NTP, Syslog, SNMP, DHCP, NAT, IP SLA)  

Todos los contenidos son **personales**, redactados como herramienta de estudio y perfeccionamiento profesional.

## Metodología de estudio

Los apuntes se basan en:

- Libros oficiales ENCOR (Exam Cram / OCG)  
- Laboratorios en GNS3, EVE-NG, CML y Packet Tracer  
- Pruebas reales sobre hardware Cisco (Catalyst 9300, ISR, WLC 9800)  
- Experiencia personal impartiendo cursos y administrando redes empresariales  

El objetivo no es sustituir documentación oficial, sino **complementarla con explicaciones claras y prácticas**.
